📌 Project Overview

Ethio SecurePass is a simple web-based cybersecurity tool that analyzes password strength and generates secure passwords. It provides real-time feedback to help users understand how strong their passwords are and how to improve them.

This project was built using basic web technologies to demonstrate cybersecurity awareness and practical front-end development skills.

🚀 Features
Password strength checker (Weak / Medium / Strong)
Real-time security feedback
Password improvement suggestions
Secure random password generator
Simple and user-friendly interface
🛠️ Technologies Used
HTML – structure of the application
CSS – styling and UI design
JavaScript – logic and password analysis
🎯 Purpose of the Project
The goal of this project is to promote cybersecurity awareness by helping users understand password security. It also demonstrates basic programming and problem-solving skills in web development.
🧠 What I Learned
Basic front-end web development
Logic building using JavaScript
User input validation
Cybersecurity fundamentals (password safety)
